# Líderes del Conocimiento - Club de Reporteros ITSE

Portal institucional desarrollado para el Club de Reporteros del ITSE en colaboración con SERTV.

## 🚀 Tecnologías utilizadas
- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (Vanilla)

## 🎯 Características
- Interfaz responsiva y adaptativa.
- Integración multimedia (audio y video).
- Diseño modular para noticias, shows y contenido institucional.

## 🌐 Enlace del proyecto
[Ver el sitio en vivo](https://lideresdelconocimientoitse.netlify.app/)

## 🛠️ Cómo ejecutar
1. Clona este repositorio.
2. Abre `index.html` en tu navegador.
